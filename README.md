## OpenShaders

Open-source WebGPU effects component library
Explore: [openshaders.com](https://openshaders.com)

### What is this?

A growing collection of beautiful, reusable GPU-powered effects you can plug into your projects.

### Roadmap

* React and ShadCN support (coming soon)

### Contributing

We’re looking for unique, high-quality ideas and implementations.

* Have a design philosophy? Share it
* Want to build an effect? Ship it
* Raise a PR

### Get Featured

The best effects get featured on the homepage of openshaders.com with credit to your socials.
